

## Cadastro e Listagem de Produtos

Este é um projeto simples que consiste em uma aplicação web para 
realizar cadastro de produtos e listar esses produtos cadastradis, 
utilizando Go (Golang) para o backend e HTML/CSS/JavaScript para o frontend.

## Execução

1. Olá, tudo bem? Seja bem-vindo ao meu projeto! 
   Agora que você extraiu o que tinha na pasta compactada siga os passoa
   abaixo.

2. Se estiver no 'Windows', apenas clice duas vezes no executável chamado
   'main'.

3. Se estiver no 'Linux', vá para o terminal e navegue até o diretório onde
   o arquivo 'Execute-me' está localizado e dê o seguinte comando:
   Comando:  ./main

OBS: NÃO SE PREOCUPE EM PROCURAR OS ARQUIVOS HTML'S , AO EXECUTAR O 'main'
     O CÓDIGO IRÁ LIGAR O SERVIDOR E ABRIR AUTOMATICAMENTE O 'cadastro.html' PRONTO
     PARA USO.
     APÓS ISSO, É SÓ UTILIZAR O VSCODE OU A IDE DE SUA PREFERÊNCIA PARA NAVEGAR PELO
     MEU CÓDIGO.
